# smallGifts
  - 小礼物是一款面向年轻的学生朋友和上班族的购物软件，主要推送一些有创意的小物件。开发环境xcode8.1,语言`swift3.0` `纯代码`布局实现，使用的第三方框架`LSXPropertTool`,`snapkit`,`Alamofire`,`ShareSDK`,`SMSSDK`,等
  - （ps: 数据转模型中再次用到了自己写的框架`LSXPropertTool`,欢迎验证使用该框架！准备完善客服Chat这一块，目前正在准备IM聊天UI这一块,后续功能和界面完善和更改后准备上线推出,个人想法：集购物，聊天，直播，和导航，后续功能持续更新。）
